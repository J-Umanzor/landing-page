Projects

Here are some of my notable projects:

  • Happyeats: A full-stack application developed alongside 3 other group members. Happyeats is designed to help users find and share recipes. My contributions included setting up the backend with Node.js and Express, integrating a MongoDB database, and implementing key features such as user authentication and recipe management. This project taught me valuable lessons in teamwork, project management, and full-stack development.
  • Pythonpuzzle: A puzzle game made using Pygame. This project involved designing engaging and challenging puzzles, implementing the game logic using Pygame, and creating an intuitive user interface. It helped me enhance my problem-solving skills and gain a deeper understanding of game development.
  • Caesarcypher: A Caesar cipher program made in Java. This project required implementing a classic encryption technique, which involved shifting characters in the alphabet. It was a great exercise in understanding encryption basics, string manipulation, and working with user inputs in Java.
